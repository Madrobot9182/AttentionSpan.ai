Our AI model monitors your mental state in real-time across four categories:

- **Not Fatigued & Focused**
- **Fatigued & Focused**
- **Fatigued & Not Focused**
- **Not Fatigued & Not Focused**

### How It Works
During each session, your **focus level** determines how many **coins** you earn.  
The longer you stay focused, the more coins you collect.

You can then use these coins to **sell or upgrade your plants**, purchasing better ones that generate **higher base earnings** each session.

## How to Run
Go into the frontend directory.  
`cd app/frontend`  
Install Dependencies.  
`npm install`  
Start the web app.  
`npm run start:all`


## Project Structure

- app - Our front end application

- data - Data files extracted from the muse 2

- models - Trained models from PyTorch

- training - scripts to train and tune the models
